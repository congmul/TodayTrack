# Feature: Analytics

## Goal
Provide insights into task performance and consistency.

## Description
Analytics differ by project type.

---

## Habit Project Analytics

### User Story
As a user, I want to see my habit consistency, so that I can improve.

### Acceptance Criteria
- Show completion rate:
  - completed / expected
- Show missed rate
- Show streak
- Support ranges:
  - weekly
  - monthly
  - yearly

---

## Task Project Analytics

### User Story
As a user, I want to understand my task performance, so that I can manage work better.

### Acceptance Criteria
- Show:
  - completed tasks
  - open tasks
  - overdue tasks
  - due soon tasks

---

## Notes
- Analytics depends on repeat engine correctness
