# Feature: Task Filtering

## Goal
Allow users to focus on specific types of tasks.

## Description
Users can filter tasks within a project.

## User Story
As a user, I want to filter tasks, so that I can focus on what matters.

## Acceptance Criteria
- View all tasks
- Filter:
  - repeating tasks
  - non-repeating tasks
- Filters do not change data, only view

## Notes
- Filter based on existence of repeatRule
