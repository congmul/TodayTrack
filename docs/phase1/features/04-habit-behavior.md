# Feature: Habit Project Behavior

## Goal
Track consistency without pressure.

## Description
In habit projects, tasks are used to measure consistency.

## User Story
As a user, I want to track my habits without warnings, so that I can improve gradually.

## Acceptance Criteria
- Tasks can be marked completed per day
- Missed tasks are recorded automatically
- No alerts or warnings for missed tasks
- Completion rate is calculated:
  - completed / expected
- Streak tracking is supported
- If a day is missed → streak resets

## UI Expectations
- Focus on progress
- Show completion rate
- Show streak
