# Feature: Today View

## Goal
Provide a single view of what matters today.

## Description
The Today view aggregates tasks from all projects.

## User Story
As a user, I want to see today's tasks, so that I can focus on what needs attention now.

## Acceptance Criteria
- Show habit tasks expected today
- Show task-project tasks due today
- Show overdue tasks
- Respect user timezone

## UI Expectations
Sections:
- Habit tasks (today)
- Tasks due today
- Overdue tasks

## Notes
- This is a core UX feature
